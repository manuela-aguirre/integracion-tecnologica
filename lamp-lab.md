# Laboratorio LAMP - Integración Tecnológica

## 1. ¿Qué es Apache?

Apache HTTP Server es uno de los servidores web más utilizados del mundo.  
Es un software de código abierto que permite alojar sitios web y entregar páginas web a los navegadores mediante el protocolo HTTP o HTTPS.

Apache funciona escuchando solicitudes de los clientes (navegadores) y enviando como respuesta archivos como HTML, CSS, imágenes o aplicaciones web.

Características principales:
- Software libre y de código abierto
- Multiplataforma (Linux, Windows, Mac)
- Soporta múltiples lenguajes como PHP
- Muy utilizado en servidores web

---

## 2. ¿Qué es LAMP?

LAMP es un conjunto de tecnologías utilizadas para desarrollar y alojar aplicaciones web.

El acrónimo significa:

- **L** → Linux (Sistema operativo)
- **A** → Apache (Servidor web)
- **M** → MySQL / MariaDB (Base de datos)
- **P** → PHP (Lenguaje de programación)

Este conjunto permite crear aplicaciones web dinámicas que interactúan con bases de datos.

Funcionamiento básico:

1. El usuario hace una solicitud desde el navegador.
2. Apache recibe la solicitud.
3. PHP procesa la lógica del sitio.
4. MySQL guarda o consulta datos.
5. Apache envía la respuesta al navegador.

---

## 3. Diferencia entre Apache y NGINX

| Característica | Apache | NGINX |
|----------------|------|------|
| Arquitectura | Basada en procesos o hilos | Basada en eventos |
| Consumo de recursos | Mayor consumo | Más eficiente |
| Configuración | Más flexible | Más simple |
| Uso común | Hosting compartido | Alta concurrencia |
| Rendimiento | Bueno | Muy alto en tráfico grande |

Apache es muy flexible y fácil de configurar, mientras que NGINX es más rápido y eficiente cuando se manejan muchos usuarios simultáneamente.
